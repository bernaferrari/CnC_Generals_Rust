PERFORMANCE BOTTLENECKS FOUND
================================================================================

1. TEXTURE DATA CLONING
   Location: texture_system.rs, texture_loader.rs
   Issue: .to_vec() copies pixel data during GPU upload
   Impact: 10-15% slower texture loading
   Fix: 2-3 hours (use Arc<[u8]> for zero-copy)

2. RWLOCK ON GLOBAL RENDER CONFIG
   Location: config.rs
   Issue: Lock contention on every frame
   Impact: 1-2% frame time overhead
   Fix: 1-2 hours (use once_cell + copy into frame context)

3. MUTEX<VEC> FOR MESH HANDLES
   Location: render_object_system.rs
   Issue: Per-frame lock + vector reallocation
   Impact: 2-3% frame time overhead
   Fix: 2-3 hours (use SmallVec or inline array)
